Release Notes

1.1.6

Improvements
  • Now running the latest stable release of JPGraph
  • Terms & Conditions Feature
  • Installer detects if Gzip is enabled or disabled
  • Admin CP displays warning message if user tries to enable gzip when it is disabled on the server
  • HTML help documents added
  • New permission to disable post counting in selected forums
  • New permission to see hidden members using the board
  • Database password is no longer held in a standard text box, it is now in a password formatted text box in installer
  • Admin stats produces a clean MercuryBoard error if there is a missing component
  • Admin CP now asks for admin password for security
  • New Feature Topic Rating
  • New Feature Mark PM as unread
  • CP now requires changed password >= 5 characters long, same as registration
  • Removed the ability of banning or stopping Guests, or the Guest Group, from using MercuryBoard in the Admin CP

Need help with using the Admin CP?
Now you can use the Admin CP help documentation containing all the basics that you will need to know, to access this documentation go into your Admin CP and click Admin CP Help on the grey bar top right.

Still having a problem?
If you have tried looking at the readme and the FAQ but you are still having problems, try going to the official MercuryBoard support forum by clicking the support link at the top of this page.

Looking for a host?
In order to use MercuryBoard you need to have a host, this is basically a server which is connected to the World Wide Web so you can upload web pages to the internet. If you don't have a host then take a look here for a list of ones that are free
