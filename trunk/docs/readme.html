ReadMe

Welcome to the MercuryBoard Readme!

In this readme you will find all the information you need to install and upgrade your board. It will also tell you how you can support the project and shows a copy of the software license agreement which MercuryBoard uses. To use this readme just click on the link you wish to visit at the top of the page or on the side panel.

If you are about to install MercuryBoard then it might be good to look at MercuryBoard's comprehensive step by step guide by clicking on 'installation' link on the top orange panel.

If there is a problem with your board or your not to sure if you should post in our support forum, click on the 'FAQ' link on the top orange panel.

There is also a brief history of MercuryBoard up to the 1.0.0 release, if you wish to read this just click on the 'about' link on the top orange panel.

Thank you for choosing MercuryBoard! 		Need help with using the Admin CP?
Now you can use the Admin CP help documentation containing all the basics that you will need to know, to access this documentation go into your Admin CP and click Admin CP Help on the grey bar top right.

Still having a problem?
If you have tried looking at the read me and the FAQ but you are still having problems, try going to the official MercuryBoard support forum by clicking the support link at the top of this page.

Looking for a host?
In order to use MercuryBoard you need to have a host, this is basically a server which is connected to the World Wide Web so you can upload web pages to the internet. If you don't have a host then take a look here for a list of ones that are free
